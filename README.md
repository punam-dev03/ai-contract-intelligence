# 🚀 AI Contract Intelligence Platform

An AI-powered full-stack web application that analyzes legal contracts, extracts clauses, evaluates risk, and provides smart insights for better decision-making.

---

## 🔍 Overview

AI Contract Intelligence is a MERN-based platform that helps users understand complex legal contracts using Artificial Intelligence.

Users can upload PDF contracts, and the system will automatically:

* Extract important clauses
* Calculate risk score
* Identify risky terms
* Suggest safer alternatives
* Simulate financial impact

---

## ✨ Features

### 🤖 AI Features

* AI Clause Extraction
* Risk Scoring Engine
* AI Contract Chat (Ask questions about contract)
* Clause Improvement (Safer Clause Generator)
* Negotiation Simulation

### 📊 Analytics & Insights

* Clause Risk Heatmap
* Contract Complexity Score
* Financial Impact Simulation
* Risk Dashboard

### 📄 Contract Management

* Upload PDF Contracts
* View Contract Details
* Download AI Report (PDF)
* Delete Contracts (with file removal)

---

## 🧠 How It Works

1. Upload contract PDF
2. Extract text using PDF parser
3. AI processes clauses using OpenAI
4. Risk scoring algorithm calculates severity
5. Results displayed in dashboard with insights
